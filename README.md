# Kişisel Tanıtım Sayfası

Bu proje üniversite dersi kapsamında, temel HTML ve CSS bilgilerini öğrenmek amacıyla hazırlanmıştır.  
Giriş seviyesinde bir web projesidir.

## Proje Açıklaması

Bu web sayfası, bir öğrencinin kendini tanıtması için oluşturulmuştur.  
Sayfa içerisinde kısa bir tanıtım, ilgi alanları ve iletişim bilgileri yer almaktadır.  
Tasarım sade tutulmuş ve temel yapılar kullanılmıştır.

## Kullanılan Teknolojiler

- HTML5  
- CSS3  

## Nasıl Çalıştırılır?

1. Proje dosyalarını bilgisayarınıza indirin.
2. Proje klasörünü açın.
3. `index.html` dosyasına çift tıklayın.
4. Sayfa internet tarayıcısında görüntülenecektir.

## Amaç

Bu projenin amacı HTML ve CSS yapısını öğrenmek ve basit bir web sayfası oluşturmaktır.  
Eğitim amaçlı hazırlanmıştır.
